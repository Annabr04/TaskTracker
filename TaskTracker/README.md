# TaskTracker CLI (Java)

A simple command-line task manager written in Java.

## Features
- Add tasks with a description and priority
- List all tasks
- Mark tasks as completed
- Delete tasks
- Tasks are saved to a local file (`tasks.txt`) between runs

## How to Run

1. Make sure you have **Java 17+** installed.
2. Compile:

```bash
cd src
javac Task.java TaskManager.java TaskTracker.java
```

3. Run:

```bash
java TaskTracker
```

You can then follow the on-screen menu to manage tasks.
